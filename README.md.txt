# Foundation-of-AI
University Foundation of Artificial Intelligence
The course includes the following topics:
1. History of Artificial Intelligence, scientific environment 
2. Concept of agents, structures of agents, rational agent, type of environments
3. Solving problem by searching 
4. Uninformed search strategies 
5. Informed/heuristic search strategies, heuristic functions 
6. Constraint satisfaction problem 
7. Adversarial search
8. Quantifying uncertainty, probability
9. Probabilistic reasoning, Bayesian networks 
10.Decision trees
11. Perceptions, neural nets, outlook
12.Reinforcement learning I. 
13.Reinforcement learning II.
